# AL'TERA site

Статический сайт для Netlify.

## Настройки Netlify

- Build command: оставить пустым
- Publish directory: `.`

Главный файл сайта: `index.html`.

## Обновление

После подключения папки к GitHub и Netlify:

1. Вносите изменения в `index.html`.
2. Запускайте `publish-update.cmd`.
3. Netlify автоматически обновит сайт.
